## Описание проекта
Необходимо написать программу для нахождения загаданного компьютером числа за минимальное количество попыток 

### Цель: 
Оптимизация существующего кода для более быстрого получения результата.

### Задача: 
Доработать существующую программу, чтобы получать загаданное число менее чем за 20 попыток.

### Условия задачи:
* Компьютер загадывает целое число от 0 до 100
* Программа должна определить это число
* Доступна информация о том, больше или меньше загаданное число
* Среднее количество попыток при 1000 повторений

### Метрика качества: 
результаты оцениваются по среднему количеству попыток в пределах 20 повторений (в исходном коде количество посвторений было ≈ 100)

### Используемый метод:
Бинарный поиск: находим число максимум за 7 попыток в диапазоне от 0 до 100
* На каждой итерации отбрасываем половину оставшихся чисел.
* log₂(100) ≈ 6.64 → значит максимум 7 попыток достаточно, чтобы найти любое число от 1 до 100.

## Этапы работы над проектом:
* Изучение исходного кода (предоставлен авторами курса) (game_core_v2)
* Написание видоизмененного кода с использованием вышеописанного метода (game_core_v3)
* Использование функции для оценки количества попыток (score_game)

## Выводы
Модифицированный код успешно решает поставленную задачу
